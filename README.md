# Smoking Health Analysis

📌 Overview
This project explores the impact of smoking on health using Power BI.  
It analyzes patient datasets to uncover relationships between smoking habits, BMI, blood pressure, cholesterol, and organ health conditions.  

👉 This work was inspired by the **end‑to‑end healthcare data analytics project by Issac Emmanuel**, whose YouTube channel is [AnalysisWithEmmanuel](https://www.youtube.com/@AnalysisWithEmmanuel).

📂 Project Structure
- CSV datasets (`health_dataset.csv`, `Organs.csv`, `Image Dataset.csv`, `condition.csv`)

🔧 Tech Stack
- Power BI (Dashboard design & visualization)

 📊 Key Features
- Smoking Status Segmentation → Current, Former, Never smokers
- Years of Smoking & Cigarettes per Day → Risk factor analysis
- BMI vs Average BMI → Visual indicators for deviation
- Organ Health Analysis → Healthy vs Damaged conditions with linked images
- Risk Factors → Blood Pressure, Cholesterol, Family History, Alcohol Consumption

📈 Insights
- Current smokers show higher damaged organ cases, especially lungs and heart.
- Long-term smoking correlates with elevated cholesterol and blood pressure risks.
- Former smokers still carry residual risks compared to non-smokers.
- Alcohol consumption combined with smoking amplifies health deterioration.
